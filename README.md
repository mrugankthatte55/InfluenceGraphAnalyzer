# InfluenceGraphAnalyzer
A real-time influence analysis system using Kafka streaming and Neo4j graph analytics. Uses graph algorithms (PageRank, BFS, Louvain) using Neo4j GDS to rank influencers and identify more than 50
emerging communities with sub-second query latency.
Integrated FastAPI based analytics service exposing graph insights via REST APIs, enabling low latency, programmatic access to social network intelligence
